## Key features

- **12 agent types** -- Customer Support, Product Recommendation, Order Management, Marketing, Inventory, Returns, Shipping, Loyalty, Content & SEO, Sales Assistant, Analytics, and Custom
- **Multi-model support** -- Claude Sonnet, Haiku, Opus, GPT-4o, GPT-4.1, Gemini 2.5, Llama 4, DeepSeek, and AWS Bedrock models
- **Shopify MCP tools** -- Connect to Shopify Catalog and Checkout via the Model Context Protocol
- **Bring Your Own Key** -- Use your own Anthropic, OpenAI, or OpenRouter API keys
- **MCP endpoint** -- Expose your ToeBox agents to Claude Desktop, Cursor, and other MCP clients
- **SQL analytics** -- Query any connected data source and build live dashboards
